# Bind-Garments

A premium company website built with **Next.js 15 (App Router)** and **Sanity CMS**.

## Tech Stack
- **Frontend**: Next.js 15, TypeScript, Vanilla CSS
- **CMS**: Sanity.io (Studio v3)
- **Styling**: Premium custom CSS with glassmorphism and smooth transitions

## Project Structure
- `/src/app`: Next.js App Router pages and layouts
- `/src/components`: Reusable UI components
- `/src/styles`: Global and component-specific styles
- `/studio`: Sanity CMS Studio (TBD)

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Design System
The project uses a premium design system defined in `src/app/globals.css`, prioritizing:
- **Vibrant Gradients**: Indigo to Pink palettes.
- **Glassmorphism**: Backdrop blur and subtle borders for cards and headers.
- **Micro-animations**: Smooth transitions using cubic-bezier.
